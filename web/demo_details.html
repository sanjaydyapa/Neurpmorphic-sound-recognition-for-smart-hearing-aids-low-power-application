üìà Understanding the Metrics

  • Accuracy: Overall correctness - how many predictions were right out of all predictions
  • Precision: When model says "this is class X", how often is it actually class X?
  • Recall: Of all actual class X instances, how many did the model correctly identify?
  • F1-Score: Balanced measure combining precision and recall (harmonic mean)
  • Cohen's Kappa: Measures agreement beyond random chance (0.74 = substantial agreement)
  • Matthews Correlation: Comprehensive metric for multi-class classification quality
  • Weighted Averages: Metrics weighted by number of samples in each class
