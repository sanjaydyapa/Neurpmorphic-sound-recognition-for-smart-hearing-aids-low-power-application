# UrbanSound8K Dataset

⚠️ **Audio files not included in repository** ⚠️

The audio files (`urbansound8k_data/audio/`) are **not pushed to GitHub** due to size constraints (~5.7 GB).

## To Use This Project:

1. **Download the UrbanSound8K dataset** from:
   - Official source: [https://urbansounddataset.weebly.com/urbansound8k.html](https://urbansounddataset.weebly.com/urbansound8k.html)
   
2. **Extract audio files** to:
   ```
   urbansound8k_data/audio/
   ├── fold1/
   ├── fold2/
   ├── fold3/
   ├── fold4/
   ├── fold5/
   ├── fold6/
   ├── fold7/
   ├── fold8/
   ├── fold9/
   └── fold10/
   ```

3. **The metadata CSV is included** in this repository:
   - `urbansound8k_data/metadata/UrbanSound8K.csv`

## Dataset Information:

- **Total Files**: 8,732 audio files
- **Size**: ~5.7 GB
- **Format**: 4-second WAV files
- **10 Classes**:
  - Air Conditioner
  - Car Horn
  - Children Playing
  - Dog Bark
  - Drilling
  - Engine Idling
  - Gun Shot
  - Jackhammer
  - Siren
  - Street Music

## Citation:

J. Salamon, C. Jacoby and J. P. Bello, "A Dataset and Taxonomy for Urban Sound Research", 
22nd ACM International Conference on Multimedia, Orlando USA, Nov. 2014.
